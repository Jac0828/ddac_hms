# HMS Client - React Frontend

This is the React frontend for the Hotel Management System. It consumes the HMS.Api backend.

## Features

- ✅ User Authentication (Login/Register)
- ✅ Room Browsing
- ✅ Booking Management
- ✅ JWT Token Management
- ✅ Responsive Bootstrap UI

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The app will run on http://localhost:3000

## Environment Variables

Create a `.env` file in the root directory:
```
REACT_APP_API_URL=http://localhost:5272
```

## API Backend

Make sure the HMS.Api backend is running on http://localhost:5272 before starting the React app.
